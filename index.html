whats new

As of 18-04-2024

There has been a few changes to the Bible Scroll app. These have been made to improve user experience, and most of them are what you - our users - have asked for in recent feedback.

A rundown of major changes that you can experience are the following:

  • New Bible translations and versions - You can now select to view verses from either the English Standard Version, New International Version, or the World English Bible!
  • Word highlighting - words or phrases that you search for will now be highlighted.
  • Ability to navigate to the next or previous verse, by clicking a verse to expand it and then using the navigation buttons.

Some more changes have been done 'under-the-hood'. This project will continue to grow and more improvements are on the way!

Thank you so much to everyone who has supported the project and given their feedback and ideas!
